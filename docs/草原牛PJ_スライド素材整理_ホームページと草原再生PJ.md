## 3. 導入部に使う情報（ナレッジベース）

### 3-1. 現状（inherit）で押さえるべき論点
- 草原は「放置すれば維持される自然」ではなく、人の関与があって維持される二次的自然。
- 人と草原の距離が広がり、担い手不足・高齢化により維持機能が弱っている。
- 草原を失うことは、景観だけでなく、水・生態系・地域文化・生業の複合的損失につながる。

### 3-2. 未来（future）で示すべき危機感
- 30年後、草原面積が現在の約4割まで減る可能性が示されている。
- この将来像を回避するには、単発保全ではなく、経済活動と保全の接続モデルが必要。
- 2021年に30年目標と取り組み方針が設定されており、行政との接続余地が明確。

### 3-3. 導入部の1スライド要約文（たたき台）
- 「阿蘇草原は、守る対象であると同時に、地域の水・生態系・生業を支える基盤である。担い手不足が進む中、30年後の縮小リスクに対し、保全と産業を同時に成立させる実装モデルが求められている。」

## 4. 本質部に使う情報（草原牛PJの価値）

### 4-1. 公式ホームから転用しやすい訴求軸
- 「牛が牛らしく生きる」という飼養哲学（無畜舎・周年放牧・野草主食）。
- 阿蘇外輪山での放牧という地域適応性（あか牛の特性と地形適合）。
- 遊休草地の活用（使われなくなった土地を生産基盤へ転換）。
- 食の価値提案（赤身の旨み、季節で変わる風味、栄養価訴求）。
- 社会価値との両立（牛・人・草原・地域への循環便益）。

### 4-2. 行政対話向けに言い換えると強いポイント
- 情緒表現: 「草原を守る」
- 政策言語: 「草資源の循環利用による地域資源管理・持続可能畜産モデルの実証」
- 情緒表現: 「牛らしい飼育」
- 政策言語: 「低投資参入可能な放牧畜産による担い手確保と地域再生」

## 5. 30分版プレゼンへの割り当て（推奨）
- 1枚目: 面談目的（次回会議化）

- 3枚目: 30年後リスク（future要約）

- 5-7枚目: 5実践モデルの要約（3枚圧縮）

- 9枚目: 課題（市場評価軸ギャップ + 冬季飼料コスト）
