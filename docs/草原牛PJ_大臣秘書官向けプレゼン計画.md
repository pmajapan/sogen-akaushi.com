# 草原牛PJ 大臣秘書官向けプレゼン計画（精査版）

## 1. 経緯と背景（現時点）
- 草原牛PJの理念・活動等について、プレゼン実施を予定している。
- 最終アウトプットはGoogleスライド形式を想定する。
- 聞き手は「現農林水産大臣の秘書官」、話し手はPJメンバーの山田雄二氏。
- 聞き手の前提知識は「阿蘇の草原」「放牧畜産」「現状課題」「持続可能性」がほぼゼロ。
- そのため、導入（知識獲得）と本質部（PJ提案）を切り分ける二段構成が有効。

## 2. やりたいこと（今回のゴール）
- まずはプレゼンの骨子を「説明責任」と「行動変容」の2軸で設計する。
- 導入部では、ゼロ知識でも誤解なく理解できるナレッジベースを提供する。
- 本質部では、草原牛PJの価値・差別性・行政連携の具体アクションを短く明確に示す。
- 1回の面談を「知ってもらう場」だけでなく「行政として何ができるかを一緒に考える場」にする。
- 面談の着地点は「次回会議化（制度接続の検討会）」とし、初回で全結論を求めない設計にする。
- 持ち帰りアクションは「地域内畜産飼料ネットワーク構築支援策の検討依頼」を明確化する。

## 3. 整理方針（構成設計）
### A. 2本立て構成
- **導入部（ナレッジベース）**: 数十ページ想定  
  - 阿蘇草原の成立背景
  - 放牧畜産の仕組み
  - 現在の課題（担い手、採算、環境、制度接続）
  - 持続可能性の論点（生態系・地域経済・食料システム）
- **本質部（PJ提案）**: 5ページ前後想定  
  - 草原牛PJが何を解くか
  - 既存施策との接続可能性
  - 行政に期待する連携アクション
  - 面談後の次アクション

### B. メッセージ設計
- 「課題提示」だけでなく「政策接続可能な提案」に着地させる。
- 価値訴求は、情緒（地域・文化）と合理（数字・制度）を併記する。
- 専門用語は初出で必ず定義し、図解を優先する。

### C. 面談成果の定義
- 最低成果: 秘書官がPJの全体像を説明できる状態。
- 目標成果: 省内で検討可能な論点として持ち帰れる状態。
- 理想成果: 次回協議（担当課レベル）に繋がる状態。

### D. 時間設計（30分〜60分）
- **30分運用（初回想定）**  
  - 00:00-03:00 目的共有（本日のゴール: 次回会議化）
  - 03:00-13:00 導入部ダイジェスト（背景・課題・必要性）
  - 13:00-23:00 本質部5ページ（PJ提案・行政接続案）
  - 23:00-30:00 QAと次回設定（論点持ち帰り、担当課接続）
- **60分運用（深掘り想定）**  
  - 00:00-05:00 目的共有
  - 05:00-30:00 導入部（ナレッジベースを丁寧に説明）
  - 30:00-45:00 本質部5ページ
  - 45:00-60:00 QA、制度接続の論点整理、次回協議アジェンダ仮置き

## 4. スライド案（たたき台）
### 導入部（例）
1. なぜ今この話か（問題提起）
2. 阿蘇草原とは何か
3. 草原と畜産の関係
4. 現場で起きている変化
5. 放牧畜産の意義
6. 失われると何が困るか
7. 課題の構造（因果）
8. 既存政策との関係
9. 先行事例（国内外）
10. 導入部まとめ（本質部への橋渡し）

### 本質部（5ページ想定）
1. 草原牛PJの定義（Who/What/Why）
2. 解決したい課題と提供価値
3. 実装モデル（5つの実践モデル要約）
4. 行政連携案（制度・実証・広報・飼料ネットワーク）
5. 本日の相談事項と次アクション（次回会議化）

## 5. 以降の作業フロー（提案）
1. 既存資料（docx等）を精査し、ファクト・数字・根拠ソースを抽出
2. 導入部のストーリーライン確定（章立てと図表ラフ）
3. 本質部5ページを先に完成（意思決定者向け）
4. 導入部を肉付け（必要ページ数に拡張）
5. Googleスライド化（話者ノート含む）
6. 初回30分版の「エッセンス抜粋台本」と、60分版の「フル説明台本」を分けて作成
7. 次回会議アジェンダ案（制度接続検討）を初回資料の末尾に付す

## 6. ユーザー回答反映（確定事項）
1. 面談時間: 30分〜1時間（初回は30分内でエッセンス提示、疑義は2回目面談へ）
2. 面談目的: 次回会議化を優先（制度接続に向けた検討機会の創出）
3. 持ち帰り依頼: 阿蘇の持続可能な放牧畜産支援の必要性・有用性への信頼形成
4. 具体アクション要請: 地域内畜産飼料ネットワーク構築支援策の検討

## 7. docx精査で抽出した本質メッセージ（プレゼン転用用）
- 提案の核は「草原保全」と「持続可能な牛肉生産」の同時達成モデル。
- 実装基盤は、以下の5実践モデル。
  1. あか牛という選択
  2. 無畜舎・周年放牧
  3. 遊休牧野の活用
  4. 自然分娩・親子放牧
  5. 繁殖と肥育の一貫経営
- 社会的便益は、地下水涵養・野焼き支援/減災・食料自給率向上・生物多様性保全・担い手育成。
- 主課題は、市場評価軸（サシ偏重）と放牧特有コストのギャップ。
- 行政と協議したい論点は、草資源の地域内循環を担う「畜産飼料ネットワーク」支援。
