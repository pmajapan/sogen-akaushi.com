# Press / Contact ページ 要件定義書

## ページの目的

KGI「マスコミ取材のエンゲージメント向上」を達成するための専用ランディングページ。
メディア関係者が必要な情報を1ページで入手でき、取材・問い合わせにスムーズに繋がる設計にする。

## URL

`/press`

## ページタイトル・meta

- **title**: `取材・お問い合わせ｜草原あか牛`
- **meta description**: `草原あか牛への取材・講演・お問い合わせはこちら。プレスキット、高解像度写真素材、ファクトシートをご用意しています。`

---

## セクション構成

### セクション1: ヒーロー

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   Press / お問い合わせ                             │
│                                                 │
│   草原あか牛に関する取材・講演・お問い合わせを          │
│   お待ちしております。                              │
│                                                 │
└─────────────────────────────────────────────────┘
```

- 背景: 阿蘇の草原の風景写真（他ページと統一感のあるもの）
- テキスト: 白文字、中央揃え

### セクション2: プレスキット

**見出し**: プレスキット

**内容**:

| 素材 | 形式 | 内容 |
|------|------|------|
| ファクトシート | PDF | 草原あか牛の概要、数値データ、生産者情報 |
| 高解像度写真セット | ZIP | 草原風景、牛、肉、レストラン料理、生産者の写真（報道利用可） |
| ロゴデータ | ZIP | 草原あか牛ロゴ（PNG/SVG/AI） |

**ダウンロードボタン**: 「プレスキットをダウンロード」

**Wix実装方法**:
1. 各ファイルをWixメディアマネージャーにアップロード
2. ボタンにダウンロードリンクを設定
3. または Wix の「ドキュメントボタン」要素を使用

**準備が必要な素材リスト**:
- [ ] ファクトシートPDF（A4 1〜2枚程度）
  - 草原あか牛とは（100字以内の要約）
  - 数値データ（放牧面積、頭数、放牧日数等）
  - 「草原あか牛プレミアム」の説明
  - 新保吉伸氏の紹介（肩書き・経歴要約）
  - 提携レストラン一覧（店名・所在地）
  - 会社/団体概要
- [ ] 高解像度写真（最低10枚、300dpi以上推奨）
  - 阿蘇の草原風景 2〜3枚
  - 放牧中のあか牛 2〜3枚
  - 草原あか牛プレミアムの肉 1〜2枚
  - レストラン料理写真 2〜3枚
  - 生産者（牛飼い）ポートレート 各1枚
- [ ] ロゴデータ（PNG透過、SVG、できればAI形式）

### セクション3: メディア掲載実績（該当がある場合）

**見出し**: メディア掲載

過去の掲載実績をリスト or グリッド形式で表示:

```
┌──────────┐ ┌──────────┐ ┌──────────┐
│ メディア名  │ │ メディア名  │ │ メディア名  │
│ 掲載日     │ │ 掲載日     │ │ 掲載日     │
│ テーマ概要  │ │ テーマ概要  │ │ テーマ概要  │
└──────────┘ └──────────┘ └──────────┘
```

- 新保吉伸氏の NHK「プロフェッショナル 仕事の流儀」出演は必ず掲載
- 他にTV・雑誌・Web掲載があれば追加

### セクション4: お問い合わせフォーム

**見出し**: お問い合わせ

| フィールド | 種別 | 必須 | 備考 |
|----------|------|------|------|
| お名前 | テキスト | 必須 | |
| 会社名 / 媒体名 | テキスト | 必須 | |
| メールアドレス | メール | 必須 | |
| 電話番号 | 電話 | 任意 | |
| お問い合わせ種別 | ドロップダウン | 必須 | 選択肢: 取材依頼 / 講演依頼 / 商品に関するお問い合わせ / その他 |
| お問い合わせ内容 | テキストエリア | 必須 | |

**送信ボタン**: 「送信する」（プライマリボタン）

**送信後メッセージ**: 「お問い合わせありがとうございます。3営業日以内にご返信いたします。」

**Wix実装方法**:
1. 「+ 追加」>「お問い合わせ」>「お問い合わせフォーム」を選択
2. フィールドを上記仕様に合わせてカスタマイズ
3. Wixダッシュボード > 「受信箱」でフォーム送信を管理
4. 通知メール設定: 管理者メールアドレスに送信通知を設定
